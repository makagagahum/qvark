# Governance

Qorx is maintained by Marvin Sarreal Villanueva from Metro Manila.

Project contact: marvin@orin.work. Alternate contact: msarvillan@gmail.com.

The project is free software. The name, marks, `.qorx` source format, `.qorxb`
bytecode format, and `qorx://` scheme identify the official project.

## Changes

Language and protocol changes should include:

- updated handbook text;
- parser/runtime tests;
- compatibility notes;
- proof commands when behavior changes.

## Compatibility

Qorx v1 source compatibility is preserved unless a future release declares a
new language version.

## Forks

Forks are allowed under the license. Modified forks should use a distinct name
unless they are clearly presented as unofficial Qorx forks.
